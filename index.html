My Projects

E-PILKEBEM (Elektronik Pemilihan Ketua Dan Wakil BEM) STMIK PGRI TANGERANG

Aplikasi ini telah dikembangkan sejak tahun 2021 di bawah kepemimpinan Ka Dhama sebagai Ketua BEM STMIK PGRI Tangerang. Tujuan utamanya adalah menyelenggarakan pemilihan secara daring tanpa adanya pelanggaran atau kecurangan. Saat ini, aplikasi masih aktif dan terus dikembangkan oleh saya untuk memfasilitasi pemilihan Ketua dan Wakil Ketua BEM. Aplikasi ini dirancang berbasis web menggunakan teknologi PHP versi 7.4, dengan penggunaan AJAX dan Bootstrap sebagai framework CSS-nya.

Catatan Keuangan

Laundry

Aplikasi ini saya kembangkan sebagai sistem pemesanan untuk layanan laundry di Atar Laundry. Fitur-fitur yang disediakan meliputi manajemen master data jenis laundry dan pakaian, proses transaksi laundry, konfirmasi pengiriman, serta penyediaan laporan dan sistem pelacakan pengiriman atau penjemputan bagi pengguna. Sistem ini dibangun menggunakan PHP dan Bootstrap untuk memfasilitasi pengalaman pengguna yang optimal.

PPDB (Penerimaan Peserta Didik Baru) Di SMK PGRI NEGLASARI

Aplikasi Penerimaan Peserta Didik Baru (PPDB) untuk SMK PGRI Neglasari dibangun dengan menggunakan teknologi PHP dan Bootstrap. Aplikasi ini dirancang untuk mempermudah proses pendaftaran peserta didik baru secara online. Dengan menggunakan aplikasi ini, calon siswa dapat mengakses formulir pendaftaran, mengunggah dokumen yang diperlukan, serta melacak status pendaftaran mereka secara real-time. Dengan aplikasi PPDB ini, proses penerimaan peserta didik baru dapat dilakukan secara efisien dan transparan, memungkinkan sekolah untuk mengelola jumlah pendaftar dengan lebih baik dan meningkatkan pengalaman pendaftar.
